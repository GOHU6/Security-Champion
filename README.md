# Secure Expense (Spring Security Project)

Construction d'une application Java Spring Boot sécurisée qui permet aux employés d’une entreprise de saisir leurs notes de frais.

Intégration des bonnes pratiques de sécurité applicative :
  - Authentification & autorisation (JWT, RBAC)
  - OWASP
  - Audit & logs sécurité
  - Gestion des secrets
  - Tests de sécurité
  - SAST / DAST (ex : Dependency-Check, ZAP)
  - CI/CD sécurisé
  - ...
